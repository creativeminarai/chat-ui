# チャットシナリオ作成ガイド

このプロジェクトにおける、お客様の興味を引き、自然に読み進めていただくためのライティング・設計ノウハウをまとめます。

## 1. 概念をポジティブに書き換える
特定の言葉（例：酸味）がネガティブに捉えられる可能性がある場合、それを否定するのではなく、新しい価値観として提示します。

*   **Bad**: 「酸っぱいパンは苦手ですか？」(苦手な人を遠ざける、またはネガティブな記憶を想起させる)
*   **Good**: 「パンの酸味って…？」、「パンにおける酸味の概念が変わります」(好奇心を刺激し、試してみたいと思わせる)

## 2. 次のナビゲーションをスムーズにする「引き」の文章
各メッセージの最後の一文を、次の選択肢（ボタン）に繋がる「伏線」として設計します。これにより、お客様が迷うことなく次に進めます。

*   **手法**: 文章の最後を「その秘密は、実は『〇〇』にあります。」「その理由は……」といった形で締めくくる。
*   **効果**: 選択肢に「〇〇の秘密って？」と置くことで、お客様の頭の中で「答えを知りたい」という欲求と操作が直結します。

## 3. 顧客目線の選択肢（ボタン）ラベル
「〇〇について聞く」「〇〇の詳細」といった事務的なラベルではなく、お客様の心の声を代弁するような言葉を選びます。

*   **Bad**: 「仕込みのこだわりを聞く」(作り手側の都合に見える)
*   **Good**: 「熟成の秘密って？」(お客様が自然に抱く疑問を代弁する)

## 4. 情報の階層化（情報の保存と整理）
専門的なこだわり（工程、日数、材料の詳細）は非常に重要ですが、最初からすべてを提示すると情報過多になります。

*   **戦略**: 
    1. 魅力的なフック（味、香り、体験）で興味を引く。
    2. ネスト（階層）を深くし、興味を持った人だけが詳細（こだわり工程など）へ進めるように設計する。
    3. 以前の重要な情報は消さず、新しい入り口（シナリオ）を追加して繋ぎ込む。
